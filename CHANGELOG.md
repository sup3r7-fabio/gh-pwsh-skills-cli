# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Course navigation features with `next` and `back` commands
- Comprehensive release pipeline with GitHub Actions
- Cross-platform builds for Linux, macOS, and Windows
- Version information and `--version` flag support
- Automated testing and linting in CI/CD
- Development tools (Makefile, version bump script)

### Changed
- Refactored course detection logic into shared utilities
- Enhanced help text and command descriptions
- Improved error handling and user feedback

### Fixed
- Import optimization and code organization
- Syntax errors and build issues

## [1.0.0] - 2025-11-05

### Added
- Initial release of PowerShell GitHub Skills CLI Extension
- `status` command to show progress across all courses
- `hint` command for contextual PowerShell learning tips
- `validate` command to test PowerShell code locally
- Support for 4 PowerShell GitHub Skills courses:
  - Course 1: PowerShell Fundamentals
  - Course 2: Pipelines & Filtering
  - Course 3: Functions & Modules
  - Course 4: Automation & DevOps
- Interactive progress tracking with visual progress bars
- Cross-platform PowerShell validation
- GitHub repository integration
- Comprehensive documentation and examples
